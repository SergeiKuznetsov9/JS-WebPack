После создания пустого проекта (без сервера и вебпака), устанавливаем ВебПак
npm install --save-dev webpack webpack-cli

Далее запускаем команду:
npx webpack --entry ./src/sample_index.js --mode production
Эта команда говорит: смотри мой код начиная с index.js и выведи то, что готово к продакшину.

Выполнение этой команды создаст папку dist и файл main.js в ней. main.js - это уменьшенная версия
нашего кода. Этот файл можно использовать вместо index.js, все будет работать одинаково.
Но при этом кое что очень важное все же произошло. И чтобы увидеть что именно, нам необходимо 
установить некоторые зависимости.

Мы добавим две зависимости: одну - пакет, а другую — нашу собственную.
Мы будем использовать их для создания приложения для поиска произведений искусства.

1. npm install --save axios

2. Создадим файл art_helper.js и изменим index.html и index.js

Приложение по прежнему работает, но запросы не отправляются.

Далее присутпим к конфигурации webpack.config.js, создав его вручную.
После созданной конфигурации мы сможем выполнить команду
    npx webpack --config webpack.config.js
результат которой будет полностью таким же как и команды 
    npx webpack --entry ./src/sample_index.js --mode production
